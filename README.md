# Simple Upscaler

**Simple and user-friendly image upscaler running directly in the browser**

## ✨ Features

- Upscale images while preserving quality
- Works entirely in the browser — no installation required
- Clean and minimalistic interface
- Fast and easy to use

## 🚀 How to Use

1. Go to the website: [https://alexvive.github.io/simpleupscaler](https://alexvive.github.io/simpleupscaler)
2. Upload your image
3. Choose the upscale factor
4. Click Process
5. Download the enhanced image

## 🛠 Technologies

- **HTML5, CSS3, JavaScript (Vanilla JS)**
- Canvas API for image processing
- File API + Drag & Drop support
- Fully responsive design
